# CRUD API for Assignments

A simple CRUD API built with Node.js and Express that allows students to create, read, update, and delete assignment documents.

## Features
- Create new assignment entries
- Read/fetch assignment data
- Update existing assignments
- Delete assignments

## Tech Stack
- Node.js
- Express.js
- Supabase (for database)
- Cors
